# The Lagom — платформа онлайн-бронирования и продажи дополнительных услуг

## Описание проекта

Pet-проект по системному анализу для базы отдыха The Lagom.

Цель проекта — повышение конверсии бронирования, увеличение среднего чека и автоматизация процессов управления дополнительными услугами.

Проект включает анализ бизнес-проблем, проектирование целевого решения, моделирование бизнес-процессов, структуры данных и пользовательских сценариев.

---

## Бизнес-проблема

Текущий процесс бронирования имеет ряд недостатков:

- пользователи не получают полной информации об услугах;
- дополнительные услуги слабо участвуют в продажах;
- отсутствует персонализация предложений;
- часть процессов выполняется администраторами вручную;
- отсутствует аналитика пользовательского поведения.

---

## Цели проекта

- увеличить средний чек бронирования;
- повысить конверсию сайта;
- автоматизировать продажу дополнительных услуг;
- сократить ручную работу администраторов;
- улучшить пользовательский опыт.

---

## MVP проекта

В рамках MVP реализуются:

- онлайн-бронирование домиков;
- каталог дополнительных услуг;
- upsell во время бронирования;
- личный кабинет гостя;
- управление бронированиями;
- интеграция с платежной системой;
- панель администратора.

---

## Функциональные требования

- пользователь может выбрать даты проживания;
- система отображает доступные домики;
- пользователь может добавить дополнительные услуги;
- система рассчитывает итоговую стоимость;
- администратор может управлять услугами и бронированиями;
- система отправляет уведомления пользователям.

---

## Нефункциональные требования

- время ответа системы — не более 2 секунд;
- адаптивность для мобильных устройств;
- безопасная обработка платежей;
- отказоустойчивость в период высокой нагрузки.

---

## Артефакты проекта
