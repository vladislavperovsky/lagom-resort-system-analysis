# 1. Введение

## 1.1 Цель

Данная спецификация требований описывает функциональные и нефункциональные требования к веб-платформе онлайн-бронирования и продажи дополнительных услуг базы отдыха The Lagom версии 2.0.
Документ предназначен для команды разработки, тестирования и других участников проекта, задействованных в реализации и сопровождении системы.
Если не указано иное, все требования, описанные в документе, относятся к версии 2.0 и имеют высокий приоритет.

## 1.2 Глоссарий

В документе используются следующие обозначения:

- FR — функциональное требование;
- NFR — нефункциональное требование;
- MVP — минимально жизнеспособная версия продукта;
- Пользователь — гость базы отдыха, использующий систему бронирования;
- Администратор — сотрудник базы отдыха, управляющий бронированиями и услугами.

Термин «система» используется для обозначения веб-платформы онлайн-бронирования The Lagom.
